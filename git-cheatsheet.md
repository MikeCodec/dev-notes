# Git Cheatsheet

## Configuración

git config --global user.name "Mikecodec"
git config --global user.email "correo"

---

## Crear repositorio

git init

## Conectar con GitHub

git remote add origin https://github.com/USUARIO/dev-notes.git

## Subir cambios

git add .
git commit -m "primer commit"
git branch -M main
git push -u origin main
